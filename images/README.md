## Image Guidelines:

- **Aspect Ratio**: Square (1:1) images work best for the gallery grid
- **Resolution**: High quality (800x800px minimum, 1200x1200px optimal)
- **Format**: JPG for photos (smaller file size), PNG for images with transparency
- **Optimization**: Compress images to keep page load times fast
- **Naming**: Use descriptive names with hyphens (e.g., french-manicure.jpg)

## Tips for Best Results:

- Use well-lit, professional-looking photos
- Consistent background or styling across all images
- Show clear details of the nail art
- Consider adding a watermark with your logo/name
- Keep file sizes under 500KB for faster loading

The website will automatically create placeholder images if these files are missing, but real photos will make your site look much more professional!
